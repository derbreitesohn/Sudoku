# Sudoku Solver (DFS & Graph)

## About The Project
This project is a Sudoku solver written in C#. It uses a Depth-First Search (DFS) algorithm with backtracking and a custom Graph data structure to handle the game rules and constraints.

## Getting Started 

### Prerequisites
* You need the [.NET SDK](https://dotnet.microsoft.com/download) installed.

### Installation
1. Clone the repository or download the files.
2. Open your terminal in the project folder.

### Usage
Run the program with:
```powershell
dotnet run
```
The application will load three Sudoku puzzles (Easy, Medium, and Hard) and show the solution for each.
